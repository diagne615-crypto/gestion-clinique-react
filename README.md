# 🏥 Application de Gestion de Clinique

## 📖 Description

Cette application web a été développée avec **React.js** afin de faciliter la gestion d'une clinique.

Elle permet de gérer les patients, les médecins, les rendez-vous ainsi qu'un tableau de bord statistique.

---

## 🚀 Fonctionnalités

### 👤 Gestion des Patients
- Ajouter un patient
- Modifier un patient
- Supprimer un patient
- Rechercher un patient
- Consulter la fiche d'un patient
- Exporter la liste des patients en PDF

### 👨‍⚕️ Gestion des Médecins
- Ajouter un médecin
- Modifier un médecin
- Supprimer un médecin
- Rechercher un médecin

### 📅 Gestion des Rendez-vous
- Planifier un rendez-vous
- Modifier un rendez-vous
- Annuler un rendez-vous
- Rechercher un rendez-vous
- Filtrer les rendez-vous
- Exporter la liste des rendez-vous en PDF

### 📊 Tableau de bord
- Nombre total de patients
- Nombre total de médecins
- Nombre total de rendez-vous
- Nombre de rendez-vous du jour

### 🎁 Bonus
- Authentification
- Gestion des rôles
- Mode sombre
- Notifications (React Toastify)
- Export PDF
- Recherche et filtres

## 📦 Installation

Cloner le projet

```bash
git clone https://github.com/VOTRE-NOM-UTILISATEUR/gestion-clinique-react.git
```

Se déplacer dans le dossier

```bash
cd gestion-clinique-react
```

Installer les dépendances

```bash
npm install
```

Lancer le projet

```bash
npm run dev
```

---

## 📚 Dépendances installées

```bash
npm install react-router-dom
npm install react-toastify
npm install jspdf jspdf-autotable
```

## 🌐 Gestion du projet avec Git

Initialiser Git

```bash
git init
```

Ajouter tous les fichiers

```bash
git add .
```

Créer le premier commit

```bash
git commit -m "Premier commit - Application Gestion de Clinique"
```

Relier le dépôt local au dépôt GitHub

```bash
git remote add origin https://github.com/VOTRE-NOM-UTILISATEUR/gestion-clinique-react.git
```

Renommer la branche principale

```bash
git branch -M main
```

Envoyer le projet sur GitHub

```bash
git push -u origin main
```

Après chaque modification

```bash
git add .

git commit -m "Description de la modification"

git push
```

Vérifier l'état du dépôt

```bash
git status
```

Afficher le dépôt distant

```bash
git remote -v
```

Afficher la branche actuelle

```bash
git branch
```
